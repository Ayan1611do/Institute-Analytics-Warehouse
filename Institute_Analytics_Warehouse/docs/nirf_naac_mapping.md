# NIRF and NAAC Metrics Mapping

## Overview

The Institute Analytics Data Warehouse includes quality-related metrics that support institutional performance monitoring. These metrics are inspired by accreditation and ranking frameworks such as NIRF (National Institutional Ranking Framework) and NAAC (National Assessment and Accreditation Council).

---

## Quality Metrics Included

### Student Strength

Measures the total number of enrolled students in the institution.

Data Source:

* students table

Purpose:

* Academic planning
* Capacity analysis

---

### Student Performance

Measures student academic achievement based on examination results.

Data Sources:

* exam_results table

Indicators:

* Average Marks
* Pass Percentage

Purpose:

* Academic quality monitoring

---

### Attendance Analysis

Measures student engagement and participation.

Data Source:

* attendance table

Indicators:

* Average Attendance
* Department-wise Attendance

Purpose:

* Student performance tracking

---

### Placement Performance

Measures employability outcomes.

Data Source:

* placements table

Indicators:

* Placement Count
* Average Package
* Company-wise Placements

Purpose:

* Career outcome analysis

---

### Faculty Strength

Measures academic resources available.

Data Source:

* faculty table

Indicators:

* Total Faculty
* Department-wise Faculty Distribution

Purpose:

* Academic resource assessment

---

## NIRF Alignment

The project supports the following NIRF-related areas:

* Teaching, Learning and Resources
* Graduation Outcomes
* Student Strength
* Faculty Strength
* Placement Statistics

---

## NAAC Alignment

The project supports the following NAAC-related areas:

* Student Performance
* Teaching Learning Process
* Institutional Effectiveness
* Student Progression
* Quality Assurance Metrics

---

## Conclusion

The quality metrics implemented in this project provide a simplified representation of institutional analytics aligned with NIRF and NAAC evaluation frameworks. These metrics help administrators monitor academic performance, placements, attendance, and institutional effectiveness.
